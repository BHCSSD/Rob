
> [!NOTE]
> # INFO DUMP: Control Systems in Robotics
> 
> ## 1. Open-Loop Control
> 
> In an **Open-Loop** system, the controller sends a command to the robot without receiving any feedback. The robot executes an instruction without checking if the goal was actually achieved.
> 
> * **Real-World Example:** A standard microwave. If you set it for 2 minutes, it runs for 2 minutes. It does not know if the food is cooked; it simply follows the timer.
> * **Use Case:** Simple, predictable tasks.
> * **Weakness:** The robot cannot correct for errors (like a wheel slipping) or adapt to environmental changes.
> 
> ## 2. Closed-Loop Control
> 
> **Closed-Loop Control** (feedback control) uses sensors to adjust behavior dynamically. The robot constantly compares its **current state** against its **desired state** and adjusts to bridge the gap.
> 
> * **Real-World Example:** A self-driving car. It uses cameras to see lane lines. If it drifts, the sensors detect the error, and the controller adjusts the steering to center the vehicle.
> * **Our Objective:** We will implement **discrete control** (On/Off logic) to process sensor data for precise robot actions.
>

## Level 2: On-Off Closed-Loop Control

On-off control is a binary feedback system.

> [!NOTE]
> **Example**
>
> Think about a thermostat, it has an on-off controller. 
> If the temperature is below the target temperature, the thermostat turns on the heater. If the temperature is above the target temperature, the thermostat turns off the heater.
> 
> The sensor is checking to see if the action is complete. 
>
> * If Temp < Target: Heater ON.
> * * If Temp > Target: Heater OFF.

### The Deadband (Tolerance)

If we tell a robot to stop at *exactly* 10cm, it will jitter (oscillate) because the sensor might read 9.99cm one second and 10.01cm the next. 
To fix this, we create a **Deadband** (a range where the robot stays still).

> [!TIP]
> **Logic Formula:**
> * Too close? (`distance < (target - tolerance)`): Move Backwards.
> * Too far? (`distance > (target + tolerance)`): Move Forwards.
> * In the zone?: Stop.
>
